ATGO Connect

Cloud attendance for ZKTeco devices, in your Odoo

Stop fighting with static IPs and port forwarding. Point your ZKTeco device at ATGO Cloud, then let Odoo pull punches every 5 minutes. No inbound network access required.

How it works

1. Device → ATGO Cloud

ZKTeco devices push punches to adms.atgo.io (or the short URL you copy from the ATGO portal during device setup). Works with any ZK device that supports ADMS / Cloud Server config.

2. Odoo → ATGO Cloud

This module polls ATGO every 5 minutes via API key, creates hr.attendance, and acks logs so they aren't fetched twice. Pure outbound — no port forwarding on the Odoo host.

Features

  • Pull attendance from ATGO Cloud → Odoo hr.attendance
  • Map ZKTeco device PIN to hr.employee.barcode
  • Optional one-way employee push (Odoo → ATGO)
  • Per-device sync status, error visibility, manual re-sync
  • Compatible with Odoo 16.0, 17.0, 18.0, 19.0
  • Biometric data is dropped at the ATGO edge — never reaches Odoo

Setup in 60 seconds

  1. Install this module from the Odoo Apps store.
  2. Sign up at atgo.io (free for 1 device, unlimited employees).
  3. In ATGO portal: Settings → API keys → Generate. Copy the key.
  4. In Odoo: ATGO → Settings. Paste workspace slug + API key.
  5. Click Test connection. Cron picks up new punches every 5 minutes.
